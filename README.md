# Nyota Youth Empowerment Loan System

A complete loan application system for youth empowerment in Kenya with M-Pesa integration.

## Features

- ✅ Multi-page loan application flow
- ✅ Mobile-responsive design
- ✅ M-Pesa payment simulation
- ✅ Loan eligibility checking
- ✅ Real-time loan options
- ✅ Secure form handling
- ✅ Local storage for user data
- ✅ Vercel serverless backend

## Pages

1. **Home Page** - Introduction and CTA
2. **Eligibility Form** - User details collection
3. **Loan Options** - Select loan amount
4. **Confirm Loan** - Review and confirm
5. **Processing** - Payment processing animation
6. **M-Pesa Payment** - PIN entry simulation

## API Endpoints

- `GET /api/health` - Health check
- `GET /api/loan-options` - Get available loan options
- `POST /api/submit` - Submit loan application
